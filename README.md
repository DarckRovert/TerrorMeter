# TerrorMeter
Un medidor de daño ligero y eficiente, diseñado específicamente para los miembros del clan **El Séquito del Terror**. El registro de combate se analiza de manera independiente del idioma y funciona en clientes basados en 1.12 (vanilla) y 2.4.3 (burning crusade).  Este addon está optimizado para ofrecer un seguimiento de daño simple y rápido.
